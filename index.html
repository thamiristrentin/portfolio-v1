Sou desenvolvedora web e estudante de Análise e Desenvolvimento de Sistemas, atualmente no 3º semestre na Unicid. Trabalho com criação e design de sites, sempre buscando unir funcionalidade e estética. No meu processo, utilizo ferramentas como Photoshop, Figma e VS Code para garantir que cada projeto seja bem estruturado e visualmente atraente. Gosto de manter as coisas simples e eficientes, focando em resultados práticos e objetivos.

VS Code

Um editor de código eficiente, que facilita a programação de websites, oferecendo suporte para várias linguagens e extensões que otimizam o desenvolvimento.

Figma

Ferramenta colaborativa de design para criar interfaces de usuário e protótipos interativos, permitindo visualizar e ajustar o fluxo do site de maneira ágil.

Photoshop

Ideal para criação e edição de gráficos, layouts e elementos visuais para sites, garantindo que o design tenha impacto e profissionalismo.
